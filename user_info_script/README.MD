# 🧑‍💻 User Info Script

Этот Bash-скрипт позволяет получить подробную информацию о пользователе из системного файла `/etc/passwd`.

---

## 📋 Что делает скрипт?

После запуска скрипта ты вводишь имя пользователя, и он выводит:

- 👤 Имя пользователя
- 🔐 Пароль (обычно `x`)
- 🆔 UID (User ID)
- 🧑‍🤝‍🧑 GID (Group ID)
- 📝 Полное имя (GECOS)
- 🏠 Домашнюю директорию
- 💻 Shell по умолчанию

---
